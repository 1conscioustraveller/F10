How the Lightbox Works

The lightbox is a common UI pattern that displays images in a modal overlay. Here's how it works:

<div class="lightbox" id="lightbox">
  <button class="close" id="lightboxClose">×</button>
  <button class="prev" id="lightboxPrev">❮</button>
  <div class="lightbox-content">
    <img id="lightboxImg" src="" alt="Expanded view">
    <div class="lightbox-caption" id="lightboxCaption"></div>
  </div>
  <button class="next" id="lightboxNext">❯</button>
</div>

Key Components:

  • Overlay: The lightbox covers the entire screen with a semi-transparent background
  • Image Container: Displays the enlarged image with proper sizing
  • Navigation: Previous/Next buttons to cycle through images
  • Close Button: Allows users to exit the lightbox view
  • Caption Area: Shows additional information about the image

When a user clicks on a gallery image, JavaScript populates the lightbox with the corresponding image and caption, then displays the lightbox by adding an "active" class.
